simple-icon-toggle

simple-icon-toggle is an Angular material style component to replicate an iOS style switch.
I created this component for a web app where I needed a similar component.
In the first versions, in fact, Angular Material's library didn't include yet a similar component.
Of course now I suggest you to use the official one, but it's still available on npm for all the people who downloaded it.
Check on GitHub! https://github.com/webilend/simple-icon-toggle
